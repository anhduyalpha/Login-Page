The repository is a single npm project (`aether-auth`) that owns both the frontend app (React 19 + Tailwind CSS v4 via `@tailwindcss/vite`, served by Vite on port 3000) and the E2E tooling under `scripts/`. The root `package.json` exposes unified commands: `npm run dev` boots the Vite server, `npm run build` produces the production bundle, and `capture-screenshots` / `verify_strict_firebase` drive headless Chromium (Puppeteer) against that same dev server to capture PNGs into `docs/screenshots/` and assert Firebase/Firestore behavior. `index.html` is the single Vite entry mounting `/src/main.jsx`; there is no separate backend or API layer — the app talks directly to Firebase, and the scripts validate it in-process.
